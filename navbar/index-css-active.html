active:

زمانی مطابقت دارد که یک مورد توسط کاربر فعال می شود، برای مثال روی آن کلیک می شود.

شبه کلاس :active CSS نشان دهنده عنصری (مانند یک دکمه) است که توسط کاربر فعال می شود. هنگام استفاده از ماوس، "فعال سازی" معمولاً زمانی شروع می شود که کاربر دکمه اصلی ماوس را فشار می دهد.

شبه کلاس :active معمولاً روی عناصر a و button استفاده می شود. سایر اهداف متداول این شبه کلاس شامل عناصری است که در یک عنصر فعال وجود دارد، و عناصری را تشکیل می دهند که از طریق label مرتبط با آنها فعال می شوند.

سبک های تعریف شده توسط شبه کلاس :active توسط هر شبه کلاس مرتبط با پیوند بعدی (:link، :hover، یا :visited) که حداقل ویژگی برابری داشته باشد، لغو می شوند. برای استایل دهی مناسب به پیوندها، قانون :active را بعد از سایر قوانین مرتبط با پیوند، همانطور که توسط LVHA-order تعریف شده است قرار دهید: :link — :visited — :hover — :active.
