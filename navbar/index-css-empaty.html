empty:

عنصری را نشان می دهد که هیچ فرزندی به جز شخصیت های فضای سفید ندارد.

شبه کلاس :empty CSS نشان دهنده هر عنصری است که فرزندی ندارد. کودکان می توانند گره های عنصر یا متن (از جمله فضای خالی) باشند. نظرات، دستورالعمل‌های پردازش و محتوای CSS تأثیری بر خالی در نظر گرفتن یک عنصر ندارند.

توجه: در Selectors Level 4، شبه کلاس :empty به شکل :-moz-only-whitespace تغییر کرد، اما هیچ مرورگری در حال حاضر این را پشتیبانی نمی کند.
